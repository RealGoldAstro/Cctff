Quick Start

Get up and running in three simple steps

1

Clone or Download

Get the framework from the GitHub repository

git clone https://github.com/RealGoldAstro/Vault.git
2

Install in Studio

Place the framework in ReplicatedStorage

ReplicatedStorage > Vault > [Framework]
3

Start Building

Require the framework and start using the API

local CCTFF = require(ReplicatedStorage.Vault.CCTFF)
